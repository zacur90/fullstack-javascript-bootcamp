# TaskFlow

Aplicación de gestión de tareas desarrollada en JavaScript moderno.

## Funcionalidades

- Programación orientada a objetos
- Eventos y manipulación del DOM
- Async/await
- Consumo de API externa (JSONPlaceholder)
- Uso de localStorage
- Contador regresivo
- Notificaciones con setTimeout

## Tecnologías usadas

- HTML
- CSS
- JavaScript ES6+
