# API Usuarios - Módulo 7

## Descripción
API REST creada con Node.js, Express y PostgreSQL usando Sequelize.

## Instalación
npm install

## Ejecutar
npm run dev

## Endpoints

GET /usuarios  
POST /usuarios  
PUT /usuarios/:id  
DELETE /usuarios/:id

## Tecnologías
- Node.js
- Express
- PostgreSQL
- Sequelize
